## Sovelluksen asennus

Kopioi Git-projekti omalle koneellesi:
```
$ git clone git@github.com:macabre-cs/ohtu-miniprojekti.git 
```

Asenna poetry

```
$ poetry install
```

Siirry virtuaaliympäristöön
```
$ eval $(poetry env activate) 
```

Käynnistä sovellus

```
$ python3 src/index.py
```
